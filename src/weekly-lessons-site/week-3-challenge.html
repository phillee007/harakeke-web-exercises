Week 3 - Styling with CSS

Recap on last week

  • Html elements - an HTML element is defined by a start tag, some content, and an end tag.
  • Some elements don't have content, and don't require a closing tag
  • Don't forget your closing tags!
  • Html attributes - key/value pairs that provide additional information about an element. Always specified in the start tag.
  • Don't forget to use quotation marks!

<!DOCTYPE html>
<html>
    <title>HTML Tutorial</title>
    <body>
        <h1>I am an html element</h1>
        <h2>So am I</h2>
        <p>I am a paragraph. 
            <a href="http://some-site.com">I am a link inside a paragraph</a>
        </p>
        <img src="i-have-no-closing-tag.jpg">
    </body>
</html>

Making your site look good with CSS

  • HTML is used to structure a page and provide content.
  • CSS is the language we use to style an HTML document (web page).
  • w3schools.com

Let's look at some examples

  • Basic syntax of a CSS rule
  • Different kinds of selectors
  • Challenge - create three paragraphs using the element selector:
    1. Left-aligned, with green text
    2. Centre-aligned, with red text
    3. Right-aligned, with blue text
  • Challenge - create three headings using the class selector:
    1. Bold, with red text
    2. Underlined, with yellow text
    3. Italicised, with orange text
  • Challenge - create three links using the id selector:
    1. Bold, with red text
    2. Underlined, with yellow text

  • Links with the <a> element
    1. Find a link
    2. Add an <a> tag and set the "href" attribute
    3. Set the "title" and "alt" attributes

    1. Find an image
    2. Download it and save in Koder
    3. Add an <img> tag and set the "src" attribute
    4. Notice that the <img /> tag is self-closing?
  • Lists with the <ul> and <ol> elements
